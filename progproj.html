The Leak

As part of the course DIT466 Technology-driven experimental game design, a two week task was produce a small proof-of-concept game using some kind of realtime physics simulation in Unity. This resulted in us producing a simple 2D platformer, where the player must carry a bowl of simulated liquid across the level. We scored max grades for the project 50p/50p, and I for the course at large; 65p/65p.

David Campos Rodríguez was responsible for the liquid simulation and I was responsible for everything else.

Heart Raiser

This resulted in us producing the two-player racing game Heart Raiser, where the speed of one's car is correlated to one's heart rate. This project recieved the highest mark: 5/5.

This project was done together with: Petra B., David Campos Rodríguez, Ameya Kanojia, & Sabrina Samuelsson.

3Buttons

3Buttons is a short platformer game written by me in Unity C#. V1.0 was developed during the summer of 2018. This version was later ported to Android and published by the American publishing company Game Mechanic Studios for the Chinese market through Waysun. A V2.0 of the game was developed during the summer of 2019 and can be played for free on Steam. The game features three campaigns, each with five levels. Whenever a player finishes a campaign, their run is saved to the public highscores database. (Code related to 3Buttons will not be made public.)
Links: Steam Page V1.0 Official Trailer V2.0 Official Trailer Official Subreddit

Particle System & Collision Detection Demo

During the course DAT205 - Advanced Computer Graphics in 2019, each student was supposed to implement self chosen graphics related features into a small demo. This demo was written in C++ and OpenGL/GLSL. The particle system was able to both feature simple pixel particles and blending of textures to simulate jet engine exhaustion. Also, a uniform grid spatial data structure was implemented to speed up collision detection. The implementation was limited to 2D space but could handle any sized objects.
Links: Github Repository

Student Living GBG
