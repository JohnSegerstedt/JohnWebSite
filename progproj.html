3Buttons

3Buttons is a short platformer game written by me in Unity C#. V1.0 was developed during the summer of 2018. This version was later ported to Android and published by the American publishing company Game Mechanic Studios for the Chinese market. A V2.0 of the game was developed during the summer of 2019 and can be played for free on Steam. The game features three campaigns, each with five levels. Whenever a player finishes a campaign, their run is saved to the public database. As of 2020-02-13, almost a hundred individual runs have been registered.
Links: Steam Page V1.0 Official Trailer V2.0 Official Trailer Official Subreddit

Particle System & Collision Detection

During the course DAT205 - Advanced Computer Graphics in 2019, each student were supposed to implement self chosen graphics related features into a small demo. This demo was written in C++ and OpenGL/GLSL. The particle system featured both Both for simple pixel particles and for blending textures together to simulate jet engine exhaustion. Also, a uniform grid spatial data structure was implemented to speed up collision detection. The implementation was limited to 2D space but could handle any sized objects.
Links: Github Repository

Student Living GBG

This was a join project between me, Peter Gärdenäs, Jonathan Gildevall, and Erik Magnusson during the course TDA367 - Object-oriented programming project in 2017. We developed an Android app, able to combine all posted student accommodations from both SGS Studentbostäder and Chalmers Studentbostäder. The app had features such as being able to favorite certain search critera and to notify the user when a housing was posted that matched the given criteria.
