3Buttons

3Buttons is a short platformer game written by me in Unity C#. V1.0 was developed during the summer of 2018. This version was later ported to Android and published by the American publishing company Game Mechanic Studios for the Chinese market. A V2.0 of the game was developed during the summer of 2019 and can be played for free on Steam.

Student Living GBG

This was a join project between me, Peter Gärdenäs, Jonathan Gildevall, and Erik Magnusson during the course TDA367 - Object-oriented programming project. We developed an Android app, able to combine all posted student accommodations from both SGS Studentbostäder and Chalmers Studentbostäder. The app had features such as being able to favorite certain search critera and to notify the user when a housing was posted that matched the given criteria.
Links: Github Repository Hand-in Report
